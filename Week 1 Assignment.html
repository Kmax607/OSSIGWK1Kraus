Planning a website involves several steps: defining the site’s purpose and target audience, creating a site map, designing the layout and navigation, and developing the content. It’s essential to consider user experience and accessibility throughout the planning process.

In Chapter 1, I learned about the basic structure of an HTML document, including the necessary elements like DOCTYPE, html, head, title, meta, and body. I also learned how to create a basic webpage and the importance of using proper syntax and indentation for readability.
